# Seguridad de la Información: Práctica de zk-SNARKs

## Requisitos
Entorno de desarrollo con [Node.js](https://nodejs.org/en/download/) instalado.

```bash
npm install
```

En caso de querer modificar los circuitos, es necesario tener instalado [Circom](https://docs.circom.io/getting-started/installation/) y [SnarkJS](https://github.com/iden3/snarkjs).
